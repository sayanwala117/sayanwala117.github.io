Projects

SStarty

SStarty is a Python tool that lets you replace Steam Big Picture's default boot-up video with your own WebM file, adding a personal touch to your gaming experience. With a simple setup and open-source flexibility, you can easily customize Steam’s startup animation in just a few steps.


USecDrive

USecDrive is a tool for encrypting and decrypting data for local or cloud storage, offering both a GUI and CLI. However, due to current bugs and instability, it is recommended to use it in a virtual machine or Docker container, and always keep a record of your encryption key in case of data recovery. Avoid using this program on your main machine to prevent potential data loss.


Color-to-Grayscale

The Black and White Converter is a Python program that transforms images into grayscale using linear algebra and computer vision techniques. It offers a simple interface where users can input an image path and set a desired resolution, with default HD settings for invalid inputs. The program processes the image and saves the grayscale version in the source folder.
